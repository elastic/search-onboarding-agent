# Search onboarding agent

Agent configuration files tailored to search solution onboarding

## Installation

- Run `curl -sSL https://raw.githubusercontent.com/elastic/search-onboarding-agent/main/installation-instructions | sh` to extract the markdown files and append them to your local AGENTS.md configuration.
- Ask the agent about Elasticsearch

## Development

- Update markdown files
- Run `./build` to generate a new zip file
